## Käyttö
Komennot ajetaan ohjelman juuri kansiossa.

#### Asennus
```bash
poetry install
```
#### Käynnistys
```bash
poetry run invoke start
```
#### Testaus
```bash
poetry run invoke test
```
#### Testikattavuus
```bash
poetry run invoke coverage-report
```
#### Lint
```bash
poetry run invoke lint
```
#### Format
```bash
poetry run invoke format
```
